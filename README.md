# Quiz app

# 🚀 Online Quiz Platform

A modern, futuristic, and fully responsive quiz application built using **HTML, CSS, and JavaScript**.
This project provides an interactive learning experience with animated UI, dynamic quiz handling, countdown systems, difficulty levels, and category-based question loading.

# ⚙️ How It Works

1. User selects a category
2. Chooses a topic
3. Selects difficulty level
4. Starts the quiz
5. Quiz data loads dynamically using `localStorage`
6. Questions appear one-by-one with timer
7. Final score displayed after completion
